# nvidia-app-fix
**EN:** Completely closes all Nvidia app processes that may interfere with the correct operation of the video card, but there is also the option to restart them.

**RU:** Закрывает полностью все процессы nvidia app которые могут мешать корректной работе видеокарты, но есть и возможность запустить их обратно.

**USAGE**

**EN:** Be sure to run it as administrator.
